# Für das Wohlbefinden / For well-being

![Limes Thermen - Thermal bath](/images/limes-thermen.jpg)

Fußläufig in nur fünf Minuten erreichbar ist das Besucherbergwerk Tiefer Stollen. Spannende Führungen und heilende Therapien erwarten Sie dort.
